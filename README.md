# Gaia
Création d'un reseau social

## Technologies utilisées
    - Backend (serveur) :
        * Express
        * Node
        * Javascript 
        * Base de données MongoDB

    - Frontend (client):
        - Axios
        - React        
 

### Installer l'application


1 - Installer le backend

`cd backend`

`npm install`

2 - Installer le frontend

`cd frontend`

`npm install`
